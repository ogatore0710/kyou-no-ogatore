# きょうのオガトレ（プロトタイプ）

オガトレ視聴者向けの静的Webアプリ。**サーバー無し・アカウント無し・課金無し＝運用コストゼロ設計**。

## 機能（v0.1）
- 🔍 **かたさ診断**: タップだけ5問30秒 → かたさタイプ判定（一枚板／開かずのとびら／羽根おやすみ中／棒立ちペンギン／休日のロボット／のびしろ満点さん）→ 処方箋3本（2週間コース）＋悩み別のもう1本
- 📺 **きょうの1本**: あさ/よる切替（時間帯で自動判定）・日替わりローテーション
- 🔥 **つづけた日数**: 「きょうやった！」ボタンで連続日数カウント（localStorage・端末内のみ・個人情報ゼロ）
- 出口はすべてYouTube本編＝床防衛（過去資産への送客）と同じ方向

## 技術
- 単一 `index.html`（ビルド不要・依存ゼロ・バニラJS約9KB）
- 動画データはコード内の `V` 定数（video_id は ogatore.db 由来の実データ）
- サムネイルは i.ytimg.com から直接取得

## 配備
- プロトタイプ確認: `cp index.html ~/srv/navi/` → http://100.111.99.112:8787/navi/ （Tailscale内・本人用）
- 公開時の選択肢: GitHub Pages（公開リポジトリ）／ ogatore.net にファイルを置くだけ

## 運用（＝ほぼ無し）
- 新作動画を処方に足す時: `index.html` の `V` と各リストに1行追加（alanに言えば1分）
