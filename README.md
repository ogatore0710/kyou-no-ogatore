# #きょうのオガトレ（β版）

オガトレ視聴者向けの静的Webアプリ。**サーバー無し・アカウント無し・課金無し＝運用コストゼロ設計**。

## 機能（v0.5 — 2026-07-11 更新・タブ名/件数を2026-07-15に部分更新）

### 🗨 オガトレ相談室（本人監修・サーバー不要のAI風チャット）
- からだの悩みを打つ／チップを選ぶと、**オガトレの言葉で「見立て→やるべき動画→続け方」**を吹き出しで返すチャット機能。回答は本人監修の事前パターン集（`soudan-kb.js`）から選ぶ方式で、本物のLLMは使わない＝無料・オフライン可・運用ゼロ・暴走なし
- **119インテント／聞き取り語1,000超**（悩み・状況・体型・やり方Q&A・雑談）。実際の視聴者コメント9,386件から言い回しを採掘
- **安全設計**: 激痛・しびれ・ぎっくり直後・産後・診断名などは最優先で受診案内に分岐。四十肩の炎症期など「急性期は伸ばすより受診が先」を明記。医学ファクトチェック済み（受診導線／解剖生理／過大広告の3観点）
- **ラリー対応**（「もっと短いの」「他には?」を文脈で解決）・**雑談レイヤー**（2連続でストレッチに引き戻す）・答えられない悩みは📮でリクエスト送信（回答バンクの栄養に）
- 右下の💬常駐FAB（オガトレ通信の写真FABの1段上）とホームの入口カードから起動
- 📅 **2週間プラン**: 相談の答えを「2週間プラン」にすると、ホームの「あなた用」枠がその悩み専用の日替わり動画に。完走で紙吹雪のお祝い

### 🌱 はじめてガイド（初回オンボーディング）
- 初回起動でオガトレが対話式に「体、硬いほう?→気になるのは?→いつやる派?」と聞き、チェック／相談室／今日の1本へ導く。スキップ可・使い方タブから再入場可

### そのほか今回の追加
- 📦 **記録のひっこし**: 機種変対策。記録をコピー→新端末で貼るだけ（サーバー不要）
- 🔠 **文字サイズ**: 「ふつう／大きめ」切替（50-60代向け）
- 🎉 **きょうやった！の紙吹雪**・**季節マークの通年化**（入道雲・もみじ・雪・桜・初日の出…）・**新バージョンお知らせトースト**
- 🎬 タイプ別イラスト（AI生成）・きょうの1本/せんぱいの声/処方プールの大幅拡充（動画64本・声120件）

### ホーム
- 💬 **きょうのひとこと**: 尾形さんの声かけフレーズを日替わり表示（コメント2,400件分析で「声かけ目当てに来る」文化を確認して実装）
- ⏰ **いつやる派？**: 生活アンカー設定（朝おきて／おふろ上がり／寝るまえ）→ あいさつ文が変わる。実装意図（if-thenプラン）の研究ベース
- 🔍 **かたさチェック**: 5問30秒 → 6タイプ判定 → おすすめの3本に「①まずほぐす→②メイン→③しあげ」の順番ラベル＋ペースの公式見解（週3でも効く・休む日はあってよし）
- 📺 **きょうの1本**: あさ/よる切替・日替わり・コメント欄（なかまの記録）への導線
- 🌱 **おかえり画面**: 3日以上あいて開くと、責めずに歓迎（習慣研究の知見つき）＋かたさ再チェック導線

### 継続エンジン2.0（コメント分析の最大の発見=「視聴者は連続より通算の文化」に合わせた）
- **通算日数が主役**（「◯日目」）・連続は補助表示。連続が切れても通算は消えない
- 🎫 **おやすみ券**: 月3枚自動付与・休んだ日に自動消費で連続がつながる（Duolingo streak freeze の研究ベース）
- 🏆 **節目=効果の現実タイムライン**: 3日「気がする」／4日「三日坊主超え」／7日／14日／30日／66日「習慣化記念日」／…1900日。節目には先輩視聴者の実例コメントを表示
- 💡 **停滞期の予告**: 12〜16日目・28〜34日目（日記コメントの離脱集中ゾーン）に「効いてないと感じる時期」ケアを表示
- 📝 **ひとことにっき**: 「きょうやった！」後に40字メモ → 記録カードに刻印＋マイ記録タブで読み返せる（コメ欄◯×日記文化のアプリ版）
- 📏 **とどくメーター**: 前屈セルフチェック（ひざ→すね→足首→つま先→ゆかの5段階）。じこベスト更新を祝う・推移バー・カード刻印
- 💛 **きょうのありがとう**: 1日1回押せる「いいね」の受け皿（「いいねを10回押したい」という声に応えた熱量の行き場）
- 📅 **リマインダー**: 時刻を選んでApple(ICS)/Googleカレンダー登録＋iPhoneショートカット自動化ガイド（push通知はサーバー導入時に検討）
- 記録カード: 通算日数・メモ・前屈・ありがとう累計を刻印。節目はゴールド＋紙吹雪

### β公開仕様（100人配布を想定）
- 🌙 **夜用ダークモード**: じどう（夜19時〜朝5時 or 端末がダーク設定なら暗く）／明るい／暗い の3択。マイ記録→つづける設定で変更
- 🛡 **免責と透明性**: かたさチェックに「あそび感覚のセルフチェック」明記・結果に個人差注記・フッターに「このアプリについて」（記録は端末内のみ・解析なし・医療行為ではない）
- 🩺 **言葉の調整**: 「診断」→「チェック」・「処方箋」→「おすすめの3本」（医療的な見え方を回避）
- 📦 **PWA**: sw.js によるオフラインキャッシュ＋ホーム画面インストール（https配信時に有効）
- 💪 **堅牢化**: localStorage書込み保護・videos.js欠落時もアプリ本体は動作・非公開動画の自動除外パイプライン
- 🔒 検索エンジンには載せない（noindex・β運用中）

## 根拠データ
- 視聴者コメント2,400件（人気動画16本×150件、innertube収集: `scripts/collect_comments.py` 相当はscratchpad）を4テーマで分析
- 検索語句501語・DATABANK・開脚生声集95件
- 習慣化研究（Duolingo streak/emergency reserves/Lally 2010/fresh start effect/実装意図メタ分析）

## 技術
- `index.html`＋`videos.js`＋`app-search.js`（ビルド不要・依存ゼロ・バニラJS）
- 記録は全部localStorage（端末内のみ・個人情報ゼロ・解析なし）
- カタログ再生成: `npm run catalog:update`（非公開動画チェック → ogatore.db から `videos.js` 生成 → QA）。ネット確認を飛ばすローカル検証は `npm run catalog:update:offline`
- 開発引き継ぎ・次の改善候補: [HANDOFF.md](HANDOFF.md) / [SPLIT-PLAN.md](SPLIT-PLAN.md) / [BETA-CHECKLIST.md](BETA-CHECKLIST.md) / [CATALOG-AUDIT.md](CATALOG-AUDIT.md) / [QA-REPORT.md](QA-REPORT.md) / [REQUEST-INBOX-HANDOFF.md](REQUEST-INBOX-HANDOFF.md)
