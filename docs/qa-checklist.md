# 実機チェックリスト（本人用・5分）

> これは何: 配布前に竜之介さん本人がスマホでなぞる最終確認。1台5分。
> iPhoneとAndroidそれぞれ1回ずつやるのが理想。Androidがなければ iPhoneだけでもOK（Android項目はスキップと記録）。
> ✕がひとつでもあったら配布を止めて、スクショを撮って開発側に渡す。
> 開発者向けの網羅版は [../BETA-CHECKLIST.md](../BETA-CHECKLIST.md)。

## iPhone（Safari）— 約3分

**A. はじめて開く（シークレットタブで＝新規ユーザー視点）**
- [ ] 開いて2秒でロゴとホームが出る（真っ白のままにならない）
- [ ] 「かたさチェック」と「いつやる派？」が最初に見える
- [ ] 「チェックをはじめる」→5問ぜんぶタップで進める→タイプ結果が出る
- [ ] 結果の「おすすめ3本」の動画サムネが表示される
- [ ] 「いつやる派？」でどれかを選ぶ→設問が消えて挨拶に変わる

**B. 記録の1周**
- [ ] きょうの1本のサムネをタップ→YouTubeが開く→アプリに戻れる
- [ ] 「きょうやった！」を押す→お祝いが出て「1日目」になる
- [ ] ひとことメモを書いて「メモをのこす」→保存された表示が出る
- [ ] 「記録カードを画像でのこす」→カードが数秒で出る（固まらない）
- [ ] 「保存・シェアする」→共有シートが開く→キャンセルしても変にならない
- [ ] 「きろく」タブ→カレンダーにきょうの印がある→印をタップで中身が見える

**C. ホーム画面追加（PWA）**
- [ ] 共有ボタン→「ホーム画面に追加」→黄色いアイコンが出る
- [ ] アイコンから起動→アドレスバーなしの全画面で開く
- [ ] 機内モードにして開き直す→真っ白にならず画面が出る（動画再生は不可でOK）
- [ ] 機内モードを戻す

**D. 見た目**
- [ ] きろく→つづける設定→「暗い」→全タブ回って読めない文字がない（とくに いつやる派・チェックの選択肢）
- [ ] 「明るい」に戻す

## Android（Chrome）— 約2分

- [ ] 開いて2秒でホームが出る
- [ ] かたさチェック5問→結果が出る
- [ ] 「きょうやった！」→1日目になる
- [ ] 記録カード→「保存・シェアする」で共有が開く
- [ ] 「︙」→「ホーム画面に追加」→アイコンから全画面で起動する

## 最後に1つだけ

- [ ] LINEの自分専用トークにURLを送り、LINE内で開く→上部に「Safari/Chromeで開いてね」の注意が出る
