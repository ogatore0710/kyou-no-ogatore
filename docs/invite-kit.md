### #きょうのオガトレ かんたん使い方

**■ はじめての日（あわせて2分）**

1. リンクをひらく
   ［スクショ: ホーム画面全体（かたさチェックが見える状態）］
2. 「チェックをはじめる」→ 5問タップ（30秒）
   ［スクショ: 診断1問目（写真お手本つき画面）］
3. あなたの「かたさタイプ」とおすすめ動画が出る
   ［スクショ: 結果画面（タイプ名が見える状態）］
4. 動画を1本やる → ホームの**「きょうやった！」**を押す
   ［スクショ: きょうやった！ボタン周辺］

**■ 2日目からの毎日（1分）**

アプリをひらく → きょうの1本を再生 → 「きょうやった！」
これだけ！あとはアプリが全部おぼえてます

**■ たのしみ**

- 記録カードを画像でのこせる（SNS投稿もOK）

**■ こまったら**

## ③ ホーム画面追加の手順（コピペ用）

### iPhone（Safari）

1. Safariでアプリのリンクをひらく
2. 画面下の**共有ボタン**（□に↑）をタップ
3. 下にスクロールして**「ホーム画面に追加」**をタップ
4. 右上の**「追加」**をタップ
5. ホーム画面に黄色いアイコンが出たら完成！次からはそこから開くだけ！

⚠️ LINEやインスタの中でひらいた人は、先に右下の「…」や共有から**「Safariで開く」**にしてから上の手順へ（アプリ内ブラウザだと記録が別保存になるため）

### Android（Chrome）

1. Chromeでアプリのリンクをひらく
2. 右上の**「︙」**をタップ
3. **「ホーム画面に追加」**（機種により「アプリをインストール」）をタップ
4. **「追加」**をタップ
5. ホーム画面にアイコンが出たら完成！

### なぜ追加するの？

- 記録がいちばん安全にのこる開き方だから
- 1タップで開けて毎日がラクだから
- 通知や個人情報は一切なし 気軽にどうぞ！
